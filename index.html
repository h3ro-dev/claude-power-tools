üîß Troubleshooting

Common Issues & Solutions

  • "command not found" ‚Üí Reinstall that specific tool
  • "permission denied" ‚Üí Add 'sudo' before the command
  • "port already in use" ‚Üí Close other applications or use a different port
  • Vercel asks for login ‚Üí Run 'vercel login' first
  • Claude doesn't respond ‚Üí Make sure Claude Desktop is running
